# Getting it to run
- To compile this program run 'g++ main.cpp'
- To run it type './a.out'

# In the program
- You will first be prompted for an input file. The file must already exist.
- Next you will be prompted for a key file. The file must already exist.
- Last you will be prompted for an output file. This doesn't have to exist, but will be appended to if it does.
